# Simple Web Application

This is a simple web application that allows users to input their name and age. It dynamically displays the input details on the same page after submission. The application is lightweight, responsive, and user-friendly, providing a seamless experience.

---

## **Features**

1. **User-Friendly Interface**:
   - Clean and intuitive form design for user input.
   - Modern styling with a responsive layout.

2. **Input Validation**:
   - Ensures that both "Name" and "Age" fields are filled out correctly.
   - Displays meaningful error messages for invalid or missing inputs.

3. **Dynamic Output**:
   - Displays the entered name and age dynamically below the form without refreshing the page.

4. **Accessibility**:
   - Designed with semantic HTML for better accessibility.

---

## **Technologies Used**

- **HTML**: For structuring the application.
- **CSS**: For styling the interface.
- **JavaScript**: For input validation and dynamic output.

## **Setup Instructions**

1. **Download the Project**:
   - Download the project folder or file as a `.zip` and extract it to your system.

2. **Open the Application**:
   - Double-click on the `index.html` file or open it in any modern web browser.

3. **Use the Form**:
   - Fill in the "Name" and "Age" fields.
   - Click the "Submit" button to see the results displayed dynamically.

---

## **How to Use**

1. Open the application in your browser.
2. Enter your name in the **Name** field.
3. Enter your age in the **Age** field.
4. Click the **Submit** button.
5. If the input is valid:
   - The details will be displayed below the form in a neat and organized manner.
6. If the input is invalid:
   - Error messages will be displayed, prompting the user to correct the input.
